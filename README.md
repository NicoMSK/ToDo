# ToDo list

Приложение на чистом JavaScript с адаптивным интерфейсом. Работа с DOM и пользовательскими событиями. Реализовано добавление, редактирование, удаление и отметка задач как выполненных, а также возможность восстановления последней удалённой задачи. 

Технологии:
JavaScript, HTML, SCSS, БЭМ
