<h1>To-Do на чистом JavaScript</h>

<h2>Описание</h2>
<p>Моё первое To-Do приложение, разработанное на vanilla JavaScript. Проект создан для практики работы с DOM, пользовательскими событиями и базовой логикой приложения без использования фреймворков.

Реализованы добавление, редактирование и удаление задач, отметка задач как выполненных, а также возможность восстановления последней удалённой задачи. Интерфейс адаптивный, стилизация выполнена с использованием SCSS и методологии БЭМ.</p>

<h3>Технологии:</h3>
JavaScript, HTML, SCSS, БЭМ, WebPack
